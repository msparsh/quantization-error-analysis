# Quantization Error Analysis

This repository provides code for analyzing quantization errors in neural networks using PyTorch.

## Features
- Deterministic multi-bit quantization routines
- Simple CNN model for demonstration
- Layer-wise error and sensitivity analysis

## Usage
1. Open `Code.ipynb` to explore and run the quantization analysis pipeline.
2. Dependencies: `torch`, `numpy` (install via pip if needed).
